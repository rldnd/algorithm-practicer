# Array [배열]

## 정의

연속된 메모리 공간에 순차적으로 저장된 데이터 모음

## 선언

### 파이썬

```python
arr = [1,2,3,4,5]
```

```javascript
const arr = [1, 2, 3, 4, 5];
```

## 특징

- 인덱스를 통해 각 데이터에 접근이 가능합니다.
  - **시간복잡도**
    - 조회 : O(1)
    - 삽입 : O(n) or O(1)
    - 삭제 : O(n) or O(1)
    - 탐색 : O(n)
- 각 언어별로 배열에 혹은 배열 자체로 사용할 수 있는 메서드들이 있습니다.
