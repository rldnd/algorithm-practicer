# [백준 15685 : 드래곤 커브 - G3] (<https://www.acmicpc.net/problem/15685>)

## `#구현`

## 접근

- 회전의 규칙성을 찾자
- 이전 회전 방향과 현재 회전 방향에 어떤 연관성이 있는지 확인하기

## 풀이

```
이전 방향 => 다음 방향
오른 -> 위
위 -> 왼
왼 -> 아래
아래 -> 오른
```

따라서 [0,1,2,3] -> [1,2,3,0]이 됨을 알 수 있음

해당 관계를 dict로 매핑 후 순서를 뒤집어서 계산하고 다시 더해줌

## 느낀점

- x,y가 바뀌어서 헷갈리 수 있음
- 회전과 규칙성이 중요함
